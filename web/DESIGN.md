# GetUsim 디자인 가이드 (항상 반영)

모든 화면 작업 시 아래 기준을 **항상 적용**한다.

## 폰트 & 아이콘
- **한글 폰트: Pretendard**, **영문·숫자: Montserrat** (Google Fonts / CDN)
- 아이콘: **이모지 금지**, **Font Awesome** 아이콘 라이브러리 사용

## 레이아웃 & 효과
- **벤토 레이아웃(Bento)**: 섹션을 아이폰 위젯처럼 사각형 박스로 분할 배치
- **글래스모피즘(Glassmorphism)**: 헤더·카드에 반투명 유리 느낌 + `backdrop-blur`
- **3D 틸트(Tilt)**: 카드에 마우스 오버 시 입체적으로 기우는 효과

## UX / 디테일
- **스켈레톤 UI**: 로딩 중 회색 박스 shimmer (빈 화면 금지)
- **Fade-up 애니메이션**: 스크롤 시 섹션이 아래→위로 나타남
- **모바일 햄버거**: 모바일에서 메뉴 숨김 → 햄버거 클릭 시 슬라이드 메뉴

## 웹 표준 & 성능 & 접근성
- **시맨틱 태그**: `<div>` 남용 금지, `<header> <nav> <main> <section> <footer>` 등 사용 (SEO)
- **접근성**: 스킵 내비게이션 추가, 키보드만으로 이용 가능, 웹 접근성 표준 준수
